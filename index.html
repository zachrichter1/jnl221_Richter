COVID deaths through sound

The majority of the data in this project is from John Hopkins University. The exceptions are data from China and any data before January 22nd is from the European Centre for Disease Prevention and Control. Also, government data is used for the UK and France. Automatic animation charts are used to show the change in COVID-19 cases and deaths.

The key message of this dataviz project is that the COVID-19 has greatly affected the entire world however has affected some continents and countries even more than others. This dataviz project is effective at conveying it’s new message because it doesn’t just have an animation for the world or each continent, it goes into depth and shows how specifically each country was individually affected. It is also effective because the animations use sound that increases in intensity with the worsening of the pandemic.

I believe this project is visceral because the pandemic is a grim, depressing thing that has personally affected almost every single person in some form. Many has lost loved ones or know someone who has lost a loved one. It has also changed the way in which we go about our daily lives.
